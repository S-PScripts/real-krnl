Krnl Key System — How to get a key

To use Krnl, you need to do the key system. This supports the developers of Krnl — it's how they get money. Follow the steps below:

  1. Join a Roblox game.
  2. Click on the key system method you want to use (Lootlabs or Linkvertise).
  3. Click Get Key. It will paste a link to your clipboard.
  4. Open your browser (Chrome, Safari etc.). Paste the link in the address bar and go to it.
  5. Wait for the captcha prompt (click the checkbox), then wait for verification. Click Continue and proceed.
  6. Complete the key system steps (there are two checkpoints on both methods). See details below for Lootlabs and Linkvertise.

Lootlabs

  1. You’ll see a page with several tasks. Click each task, then return to the page. You do not need to actually complete the external tasks.
  2. Each task should show a timer that counts down to 0. If a timer doesn’t appear immediately, wait — sometimes tasks finish at different times.
  3. When all task timers hit 0 they should complete. If they do not, reload the page and try again.
  4. Click the Unlock Content button, complete the captcha, and repeat to get your key.

Linkvertise

  1. Click Get Link → then click "I'm interested" → then click Learn More.
  2. Wait until the task completes — do not click "I ALREADY COMPLETED THIS STEP". This usually takes a minute.
  3. Click Continue and wait ~10 seconds to access the link. Then click Get Krnl Android Checkpoint.
  4. Click Open, complete the captcha, and repeat to get your key.

Once you can see your key: copy it, open a Roblox game, and click Verify Key. Keys last 24 hours — if a key doesn't work, generate a new one.

⚠️ KEYS ARE HWID-BASED — only the keys you generate will work for your device. Do not share keys.

Troubleshooting & tips

⚠️ IT IS NOT KRNL'S FAULT IF LOOTLABS/LINKVERTISE IS BROKEN!
  • LootLabs: Run the site on desktop mode if you're doing the key on your phone. This will give you one task.
  • LootLabs: Some tasks are broken on LootLabs. If you wait more than 120 seconds for a task, reload the page.
  • If Lootlabs tasks do not complete/a timer does not show, try installing a VPN (for example: ProtonVPN). Enable it, then reload the key system page.
  • If Linkvertise gives a 1-hour restriction, try clearing cookies, enabling a VPN, and reloading the page.
  • Open the key link in an incognito window.
  • If krnl.cat displays an error, restart the key system using the original link you copied.
  • If one browser/device doesn't work, try a different browser (Brave, Chrome, Firefox, Safari etc.) or device.
  • Use an incognito window.
