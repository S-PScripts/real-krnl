KSign Method

  1. Open KSign and click the plus (+) button.
  2. Click Import Files and add your IPA file (you might then need to rename it to remove .zip).

ESign Method

  1. Open ESign and click the ... button.
  2. Click Import and add your IPA file.
  3. Go back to Khoindvn and click Download Certificates.
  4. Go to ESign again and import the Certs zip.
  5. Unzip the Certs zip.
  6. Click the Certs-Khoindvn folder ‚Üí Click ESign Cert ‚Üí Click your ESign cert. After that, click Import Certificate Management.
  7. Click the Krnl IPA and click Import App Library.
  8. Wait until Krnl shows in Apps.
  9. Click it and click Signature. If it shows a revoked message, you can still install Krnl, but it will stop working after a while.

If you open Roblox and it gives an update pop-up without any "Not Now" option, join the Krnl Discord server and wait until the devs ping about an update for Krnl. Once they ping, follow the steps above.

When Krnl updates, uninstall Krnl, get the latest IPA, and import that again.

Install Krnl via Sideloadly (if Direct Install and all KSigns/ESigns fail)

‚ö†Ô∏è You must use Sideloadly to sideload the IPA again every time Krnl updates or when 7 days have passed since your last sign. This is because with a free Apple ID, each signed app expires after 7 days ‚Äî and Sideloadly can only re-sign via your computer.

Requirements

  • Mac running macOS 10.12 Sierra or later OR a Windows computer running Windows 7 or later
  • iPhone/iPad with iOS 7 or higher
  • Apple ID

Useful websites:

  • Sideloadly Website

Instructions:

  1. Connect your computer to your phone.
  2. Go to the website and download Sideloadly on your PC.
  3. Download the Krnl IPA.
  4. Open Sideloadly and enter your Apple ID (email).
  5. Click on IPA and select the Krnl IPA you downloaded.
  6. Click Start, enter your Apple ID password, and wait a few minutes.
  7. On your phone, Krnl should have downloaded. Go to VPN and Device Management, trust the app, then enable developer mode in Privacy and Security.
  8. After restarting, you should be able to open Krnl!

Install Krnl via SideStore

üí° Allows background app refreshing without a computer ‚Äî no need to re-sign every 7 days. You‚Äôll still need to manually re-sign new IPAs on your device whenever Krnl updates.

Useful websites:

  • SideStore Website
  • Discord Server

Requirements

  • Mac running macOS 10.15 or later OR a computer running Windows 10 or later
  • iPhone/iPad with iOS 14 or higher
  • Apple ID
  • StosVPN from the App Store
  • Wi-Fi connection

Step 1: Download Required Files

  1. Download AltServer
  2. Download SideStore.ipa

Step 2: Install AltServer

Mac

  1. Extract the downloaded AltServer ZIP file.
  2. Drag AltServer into your Applications folder.
  3. Open AltServer (right-click ‚Üí Open if warned by macOS).
  4. You‚Äôll see the AltServer icon in the macOS menu bar.

Windows

  1. Extract the downloaded AltServer ZIP file and run setup.exe to install AltServer.
  2. Make sure you have the non-Microsoft Store versions of iTunes and iCloud installed. If you have the Microsoft Store versions, uninstall them first.
  3. Download and install the required software if needed:
    • iTunes 64-bit
    • iCloud
  4. After installation, launch AltServer from the Start menu.
  5. You‚Äôll see the AltServer icon in the Windows system tray.

Step 3: Connect Your Device

  1. Plug your iPhone/iPad into your Windows PC/Mac using a cable.
  2. Unlock your device and tap Trust when prompted.
  3. Enter your passcode.
  4. Install any required software updates if prompted.

Step 4: Install SideStore

Mac

  1. Click the AltServer menu icon while holding the Alt key.
  2. Select Sideload .ipa ‚Üí your device.
  3. Choose the SideStore.ipa file you downloaded.
  4. Enter your Apple ID and password.
  5. SideStore should appear on your device after a few seconds.

Windows

  1. Run AltServer as Administrator and click the tray icon while holding the Shift key.
  2. Select Sideload .ipa ‚Üí your device.
  3. Choose the SideStore.ipa file you downloaded.
  4. Enter your Apple ID and password.
  5. SideStore should appear on your device after a few seconds.

Step 5: Trust the Developer App

  1. On your iPhone/iPad, go to Settings ‚Üí General ‚Üí VPN & Device Management.
  2. Select your Apple ID and tap Trust.

Step 6: Enable Developer Mode (iOS 16+)

  1. Go to Settings ‚Üí Privacy & Security ‚Üí Developer Mode.
  2. Turn it on and restart your device.

Step 7: Pair SideStore with Your Device

Mac

  1. Download idevice_pair
  2. Ensure your device has a passcode and is connected via cable.
  3. Move idevice_pair to Applications and open it (click Open if warned).
  4. Select your device from the drop-down menu, click Load, then click Install under SideStore.
  5. When it says "Success", your device is paired.

Windows

  1. Download idevice_pair
  2. Ensure your device has a passcode and is connected via cable.
  3. Extract the idevice_pair ZIP file, then open it by double-clicking or right-click ‚Üí Open.
  4. Select your device from the drop-down menu, click Load, then click Install under SideStore.
  5. When it says "Success", your device is paired.
‚ö†Ô∏è If you update or reset your device, you‚Äôll need to re-pair using idevice_pair.

Step 8: Install & Enable StosVPN

  1. Download StosVPN from the App Store. Make sure you are NOT using the Khoindvn Anti-Revoke DNS!
  2. Enable the StosVPN VPN when using or refreshing SideStore apps.

Step 9: Configure SideStore

  1. Open SideStore ‚Üí Settings.
  2. Sign in with the same Apple ID used for installation.
  3. Go to the Apps tab and tap the green ‚ÄúX days‚Äù button next to SideStore to refresh it.

Step 10: Using SideStore

  • Download the Krnl IPA.
  • Sideload .ipa files such as Krnl on your device with the + icon in the My Apps tab in SideStore.
  • If the Krnl app expires, refresh it in SideStore.
  • If SideStore expires, reinstall it via AltServer (don‚Äôt delete the app).
üí° Keep StosVPN enabled to allow automatic background refreshing.
