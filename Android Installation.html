Installing Krnl on Android

  • Uninstall (Krnl) Roblox
  • Download the latest version of Krnl. If you are Vietnamese, make sure to get VNG.
⚠️ WARNING: If you get VNG, your Roblox account will be region-locked to Vietnam and you will not be able to use global Roblox anymore on that account. This is irreversible.
  • Click on the download and install
  • That's all!

If you open Roblox and it gives an update pop-up without any "Not Now" option, join the Krnl Discord server and wait until the devs ping about an update for Krnl. Once they ping, follow the steps above.

Using Krnl on PC (Windows/Mac) via Emulator

If you are on a Windows or Mac computer, please download an emulator. Here are some options:

  • BlueStacks
  • LDPlayer (Windows Only)
  • MuMu Player

After installing an emulator, follow the same steps as above to install Krnl.

⚠️ If you are on BlueStacks, make sure you are using an Android 13 Beta instance (create one in Multi-Instance Manager/CTRL+SHIFT+8).
⚠️ According to the LDPlayer help base, it is recommended to disable Hyper-V on Windows machines when running LDPlayer 9. This could also apply to other emulators such as BlueStacks. Source: LDPlayer guide
