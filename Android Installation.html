Installing Delta on Android

  • Uninstall (Delta) Roblox
  • Download the latest version of Delta from https://deltaexploits.gg/.
  • Click on the download and install
  • That's all! Allow Delta to access Files so scripts can use it for settings, and auto-execute can function.

If you open Roblox and it gives an update pop-up without any "Not Now" option, join the Delta Discord server and wait until the devs ping about an update for Delta. Once they ping, follow the steps above.

Using Delta on PC (Windows/Mac) via Emulator

⚠️ If you are on an Intel Mac, you may need to use Hydrogen. This is because there are no easily available emulators for Intel Macs. HYDROGEN IS DETECTED, AND ROBLOX CAN BAN YOU FOR USING IT.

If you are on a Windows or Silicon Mac computer, you can download an emulator to use Delta. Here are some options:

  • BlueStacks (RECOMMENDED)
  • LDPlayer (Windows Only)
  • MuMu Player

After installing an emulator, follow the same steps as above to install Delta.

⚠️ If you are on BlueStacks, make sure you are using an Android 13 Beta instance (create one in Multi-Instance Manager/CTRL+SHIFT+8). If you experience crashing problems, try using a different emulator.
⚠️ According to the LDPlayer help base, it is recommended to disable Hyper-V on Windows machines when running LDPlayer 9. This could also apply to other emulators such as BlueStacks. Source: LDPlayer guide
